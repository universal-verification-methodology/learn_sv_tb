# Examples — Virtual interface wiring

Track A prompts. Browser lab **`vif-wiring`** is shipped.

## Prompts

1. Restate the core idea of **Virtual interface wiring** in one sentence.
2. In your own words, capture this beat: *Class-based testbenches need a bridge to the DUT pins — you cannot pass module ports into a class the way you pass integers. A SystemVerilog interface bundles signals and modports; a virtual interface is a class-side…*
3. Sketch interface → virtual interface → driver: who holds the instance?
4. Explain what breaks if two drivers poke the same vif signal.
