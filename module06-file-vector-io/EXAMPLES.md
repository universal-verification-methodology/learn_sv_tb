# Examples — File / vector I/O

Track A prompts. Browser lab **`file-vector-io`** is shipped.

## Prompts

1. Restate the core idea of **File / vector I/O** in one sentence.
2. In your own words, capture this beat: *Regression stimulus often lives outside the testbench source — hex files of input vectors and expected outputs that you load once and replay in a loop. SystemVerilog gives you dollar-readmemh and dollar-readmemb to…*
3. Sketch a stim/exp file format for an and2 (four rows) and how a TB would apply one row.
4. Name one pitfall of hand-editing vector files (column drift, wrong radix, …).
