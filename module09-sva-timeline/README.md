After this module you can explain and practice **SVA implication timeline** (`sva-timeline`), in the browser and/or offline.

2. Complete [CHECKLIST.md](CHECKLIST.md).
3. Optional self-check: `./scripts/module.sh 09 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/sva-timeline/index.html](http://127.0.0.1:8080/tools/sva-timeline/index.html)
