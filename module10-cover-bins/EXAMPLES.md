# Examples — Coverpoint / bins

Track A prompts. Browser lab **`cover-bins`** is shipped.

## Prompts

1. Restate the core idea of **Coverpoint / bins** in one sentence.
2. In your own words, capture this beat: *Functional coverage asks whether you exercised interesting values—not just whether the test passed. A coverpoint watches a field; bins partition that field into named buckets. Each sample increments hit counts for…*
3. Define a coverpoint with low/mid/high bins and mark which bin a sample of 5 hits.
4. Distinguish cover (observe) from assert (prove) in one sentence each.
